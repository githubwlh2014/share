# Kenwell's vim 配置 #
这是我个人的vim配置和插件目录，其中.vimrc为unix配置，_vimrc为windows下配置文件。
## 下载
    git clone https://github.com/kenwell/share.git

## 使用方式
     cd share/vim
     cp .vimrc $HOME/
     cp -r .vim $HOME/
## .tags 目录
存放的tag是c标准库生成的tag文件，配合ctags使用。
